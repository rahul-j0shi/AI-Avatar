# Environment Variables

This project requires API keys for two external services:

## Required Variables

```env
# OpenAI API Key (used for Whisper STT)
OPENAI_API_KEY=your_openai_api_key_here

# MiniMax API Key (used for LLM and TTS)
MINIMAX_API_KEY=your_minimax_api_key_here
```

## Service Configuration

| Variable | Service | Purpose |
|----------|---------|---------|
| `OPENAI_API_KEY` | OpenAI | Whisper speech-to-text model (`whisper-1`) |
| `MINIMAX_API_KEY` | MiniMax | LLM responses (`MiniMax-M2.7`) and TTS (`speech-2.8-turbo`) |

## Setup Instructions

1. Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```

2. Replace the placeholder values with actual API keys:
   - **OpenAI API Key**: Get from [OpenAI Platform](https://platform.openai.com/api-keys)
   - **MiniMax API Key**: Get from [MiniMax Platform](https://platform.minimax.io)

3. Never commit `.env` to version control
